| 状态码 | 提示内容     | 原因                           |
| :----- | :----------- | :----------------------------- |
| F101   | 登录失败     | wx.login失败，未获取到code     |
| F102   | 登录失败     | wx.login code 为空             |
| F103   | 获取数据失败 | 后台与微信服务端获取openid失败 |



| 状态码 | 提示内容       | 原因 |

| B202   | 数据库操作失败 |      |
| B203   | 未查询到openid |      |
